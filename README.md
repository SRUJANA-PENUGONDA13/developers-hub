## Run locally:

To run this project locally, write on your CLI:

```bash
  git clone "https://github.com/SRUJANA-PENUGONDA13/developers-hub.git"
```

Open your favorite code editor and run locally!

## Features 

1. Create a Post

   a. Like
   
   b. Comment & Replies

2. Edit & Delete Post

3. User Feed

   a. Sort by Date, Trending
   
4. User Profile

   a. Add Profile Picture
   
   b. Bio
   
   c. Portfolio URL

5. Follow/Unfollow

6. Explore Feed

7. Bookmark Post

8. Authentication
   
   a. Sign-up
   
   b. Login
   
   c. Logout
